Chicken Parkour

  • Originality: My game is original because it involves a chicken doing parkour while cleavers and soda bottles falling from the sky.
  • Technical Merit: I was able to implement wall jumping with my sprite. Using the game.arcade.physics.collide with a function inside.
  • Prototype Postmortem:

    I learned how to use the arcade physics engine and create multiple sprites from groups. The easiest part of making this game would be design the tile map and getting the sprites to spawn. The hardest part of making this game would be implementing the health, lives and egg collection system with a decent GUI
